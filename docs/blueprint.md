# **App Name**: Santa Tereza Vet

## Core Features:

- Agendamento via WhatsApp: Link direto para o WhatsApp para facilitar o agendamento de consultas.
- Exibição de Serviços: Exibição em grade dos serviços veterinários: Consultas, Cirurgias, Vacinação, Exames Laboratoriais, vinculando cada serviço ao WhatsApp para detalhes.
- Depoimentos de Clientes: Apresentar 3 depoimentos de clientes reais com fotos de animais de estimação e classificações por estrelas.
- Mapa de Localização: Integração com o Google Maps com endereço e horário de funcionamento destacados.
- Imagens de Animais Diversos: Seleção de imagens destacando a variedade de animais atendidos, como pássaros e coelhos, bem como cães e gatos.

## Style Guidelines:

- Cor primária: Azul Marinho (#142850) para transmitir profissionalismo e confiança.
- Cor de destaque: Amarelo Dourado (#F2C94C) usado exclusivamente para botões de chamada para ação e estrelas de avaliação para focar a atenção do usuário.
- Cor de fundo: Branco (#FFFFFF) para manter uma sensação de limpeza e espaço.
- Fonte do corpo e do título: 'PT Sans', sans-serif, combina um visual moderno e um pouco de calor ou personalidade, fácil de ler.
- Design Mobile-First com um cabeçalho fixo e um botão WhatsApp flutuante permanente no canto inferior direito para suporte imediato.
- Use um espaço em branco significativo para promover uma interface limpa e organizada.
- Ícones de linha fina personalizados em Azul Marinho para cada serviço.
